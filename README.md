Data Pipeline Alert System (Azure)

 📌 Project Overview
This project demonstrates a cloud-based data pipeline monitoring system using Microsoft Azure. The system processes data through a pipeline and generates alerts when failures or abnormal conditions occur.

 🎯 Objective
To design a scalable data pipeline that can automatically detect issues and notify users in real time.

 ⚙️ Technologies Used
- Microsoft Azure Data Factory
- Azure Monitor
- Cloud Storage (for datasets)

 🔄 Workflow
1. Data is ingested into the pipeline  
2. Data is processed using pipeline activities  
3. Pipeline execution is monitored  
4. Alerts are triggered on failure  

 🏗️ Architecture
Source → Data Factory Pipeline → Monitoring → Alert Notification

 🚨 Alert Mechanism
Azure Monitor is configured to detect pipeline failures and send alerts via email.
